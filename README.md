# brushless-testbench
Testbench para motores brushless y hélices, implementado con LPC1769

# Hardware
El hardware necesario para ensamblar el proyecto es el siguiente:
- Celda de carga 5Kg
- MCU: LPC1769
- Botones
- Conversor UART/USB
- IC: HX711.
- 1 x Potenciómetro
- Motor brushless con hélice
- ESC adecuado para el motor
- Sensor IR
